CSS Grid

El CSS GRID es una técnica que permite a los diseñadores crear diseños diferentes y complejos. Grid es uno de los nuevos valores de HTML5 la cual nos permite poner a la propiedad display de un elemento para maquetar de una manera mucho más fácil nuestras páginas web.

El CSS grid se puede utilizar para lograr muchos diseños diferentes, de igual manera se destaca por permitir dividir una página en áreas o regiones principales, definir la relación en términos de tamaño, posición y capas entre partes de un control construido a partir de primitivas HTML.

El grid permite a un autor alinear elementos en columnas y filas, al igual que las tablas, sin embargo, con CSS grid son posibles muchos más diseños y de forma más sencilla que con las tablas.

Ventajas de CSS Grid

  • Mucha más flexibilidad: Te permite controlar los elementos en las 2 dimensiones y con total libertad.
  • Menos código y menos bugs: Te permite controlar los elementos en las 2 dimensiones y con total libertad.
  • Optimización de recursos: Al ser código más simple y consistente, se economiza tanto en tiempo como en recursos necesarios para mostrar la página.
  • Responsive mas sencillo: Al ser código más simple y consistente, se economiza tanto en tiempo como en recursos necesarios para mostrar la página.

    • Conceptos basicos

    Elementos principales del Grid CSS.

  • Contenedor: El elemento padre contenedor que definirá la cuadrícula o rejilla.
  • ìtem: Cada uno de los hijos que contiene la cuadrícula (elemento contenedor).
  • Celda (grid cell): Cada uno de los cuadros que conforman la cuadrícula, esta sería la unidad mínima.
  • Area (grid area): Región o conjunto de celdas de la cuadrícula.
  • Banda (grid track): Banda horizontal o vertical de celdas de la cuadrícula.
  • Linea (grid line): Separador horizontal o vertical de las celdas de la cuadrícula.
